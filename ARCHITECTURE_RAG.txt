                    AgroIA RAG System - Arquitetura
================================================================================

COMPONENTES PRINCIPAIS
================================================================================

1. INDEXAÇÃO (Batch - Uma vez)
   indexar_agro_corrigido.py → SentenceTransformer → 216 chunks + embeddings

2. API RAG (Real-time)
   FastAPI (chat/api_rag.py) Port 8000
   - POST   /rag/buscar (busca semântica)
   - GET    /rag/stats   (estatísticas)
   - GET    /health      (health check)

3. INTEGRAÇÃO COM CHAT
   example_rag_integration.py - Exemplo de uso com LLM


FLUXO COMPLETO DE DADOS
================================================================================

Usuário pergunta
    ↓
API /rag/buscar (FastAPI)
    ↓
SentenceTransformer gera embedding da query
    ↓
Supabase: Lê 500 chunks + embeddings de pdf_chunks
    ↓
NumPy: Calcula similaridade coseno para cada chunk
    ↓
Ordena por relevância (Top-K, default K=5)
    ↓
Retorna JSON com chunks + scores de similaridade
    ↓
(Opcional) LLM processa com contexto
    ↓
Resposta final ao usuário


BANCO DE DADOS (Supabase)
================================================================================

Tabela: pdf_chunks
├─ id              : bigint (PK)
├─ licitacao_id    : bigint (FK)
├─ documento_id    : bigint (FK)
├─ processo        : text
├─ nome_doc        : text
├─ chunk_index     : int
├─ chunk_text      : text (importante!)
├─ embedding       : vector(384) ← RAG
├─ tokens_aprox    : int
└─ indexado_em     : timestamp

Índices: HNSW(embedding), processo, documento_id

COMO USAR
================================================================================

1. Iniciar API RAG:
   python chat/api_rag.py
   ou
   uvicorn chat.api_rag:app --reload --port 8000

2. Testar Busca:
   curl -X POST http://localhost:8000/rag/buscar \
     -H "Content-Type: application/json" \
     -d '{"query":"Leite para merenda escolar","top_k":5}'

3. Ver Documentação:
   Acesse: http://localhost:8000/docs (Swagger UI)

4. Integrar com Chat:
   from chat.example_rag_integration import responder_com_rag
   responder_com_rag("Sua pergunta aqui")
